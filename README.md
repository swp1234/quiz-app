# 🎯 지식 퀴즈 앱

재미있는 상식 퀴즈로 지식을 테스트해보세요!

## 🌐 라이브 데모

**https://dopabrain.com/quiz-app/**

## ✨ 주요 기능

- 📝 **100개의 다양한 퀴즈** - 매번 랜덤으로 10개 출제
- 🎨 **9개 카테고리** - 일반상식, 과학, 역사, 경제, IT, 지리, 문학, 생활, 건강
- 📊 **난이도 구성** - 쉬움(10%), 보통(50%), 어려움(40%)
- 📱 **모바일 반응형** - 스마트폰, 태블릿, PC 모두 지원
- ⚡ **PWA 지원** - 앱처럼 설치 가능
- 🎯 **실시간 점수** - 즉시 피드백

## 🛠 기술 스택

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Type:** Progressive Web App (PWA)
- **Deployment:** GitHub Pages

## 📂 프로젝트 구조

```
quiz-app/
├── index.html          # 메인 HTML
├── manifest.json       # PWA 설정
├── css/
│   └── style.css      # 스타일시트
├── js/
│   ├── app.js         # 앱 로직
│   └── quiz-data.js   # 퀴즈 데이터
├── icon-192.svg       # PWA 아이콘 (192x192)
├── icon-512.svg       # PWA 아이콘 (512x512)
└── README.md
```

## 🚀 로컬 실행

### 방법 1: 직접 열기
```bash
# index.html을 브라우저에서 직접 열기
```

### 방법 2: 로컬 서버
```bash
# Python 사용
python -m http.server 8000

# Node.js 사용
npx serve

# 브라우저에서 http://localhost:8000 접속
```

## 📱 PWA 설치

모바일에서:
1. 사이트 접속
2. 메뉴 → "홈 화면에 추가"
3. 앱처럼 사용!

## 🎮 사용 방법

1. 퀴즈 시작
2. 4개의 보기 중 정답 선택
3. 10문제 완료 후 점수 확인
4. 다시 시작하여 도전!
